
 🗿📄✂️ Rock-Paper-Scissors Game

A simple Rock-Paper-Scissors game built in Java where you play against the computer! 🎮

 📌 How to Play
1. Run the program.
2. Enter your choice: **Rock**, **Paper**, or **Scissors**.
3. The computer randomly selects its choice.
4. The winner is decided based on classic Rock-Paper-Scissors rules:
   - Rock beats Scissors🏆
   - Scissors beats Paper ✂️
   - Paper beats Rock 📄
   - Same choices result in a tie 🤝
5. See the result instantly!
